    13 ## Overview

    15 STM32F469NIH6 MCU.
    16 The MCU presents a large list of peripherals available, but specifically this development board exposes:

    45 ## Connectors and Pinout
    46 
    47 - 1 USB Mini-B used by the On-board ST-LINK debugger (Ref.CN1)
    48 - 1 USB Micro-AB Full speed (Ref. CN13)
    49 - Arduino(R) Uno expansion connectors (Refs. CN5/6/7/8)
    50 - microSD(tm) card receptacle.
    51 - Audio 3.5 mm stereo jack
    52 
    53 Through 2 connector we have available:
    54 
    55 ### CN11
    56 
    57 - I2C1
    58 - Reset
    59 
    60 Also provides power pins, 3V3 and GND
    61 
    62 ### CN12
    63 
    64 This extension connector gives access to:
    65 
    66 - CAN2
    67 - USART6 (Tx, Rx)
    68 - I2S2
    69 - SPI1
    70 - 7 timers channels
    71 - Speaker output (1W)
    72 
    73 Also provides some system signals and power.
    74 
    75 **Important!** note than some pins are shared between peripherals.

    85 ## Working with this Dev-Kit
    86 
    87 To start the **gcc-arm-embedded** toolchain have to be installed,
    88 we can follow the usual process [here](https://github.com/RIOT-OS/RIOT/wiki/Family:-ARM).
    89 
    90 ## Flashing the device
    91 
    92 ### Flashing the Board using OpenOCD
    93 
    94 The start install **OpenOCD**, also we can follow the usual process
    95 [here](https://github.com/RIOT-OS/RIOT/wiki/OpenOCD)
    96 
    97 Once everything is installed and ready to work just connect the board through
    98 the USB Mini-B connector try to compile and flash some code, type:
    99 
    100 ```
    101 make flash BOARD=stm32f469i-disco
    102 ```
    103 
    104 ### Flashing the Board Using ST-LINK Removable Media
    105 
    106 On-board ST-LINK programmer provides via composite USB device removable media.
    107 Copying the HEX file causes reprogramming of the board. This task
    108 could be performed manually; however, the cpy2remed (copy to removable
    109 media) PROGRAMMER script does this automatically. To program board in
    110 this manner, use the command:
    111 ```
    112 make BOARD=stm32f469i-disco PROGRAMMER=cpy2remed flash
    113 ```
    114 @note This PROGRAMMER was tested using ST-LINK firmware 2.37.26. Firmware update
    115 s
    116 could be found on [this STM webpage](https://www.st.com/en/development-tools/sts
    117 w-link007.html).
