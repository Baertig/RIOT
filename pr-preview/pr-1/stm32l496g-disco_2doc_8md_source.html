    5 ## Overview
    6 
    7 The ST [STM32L496G-DISCO](https://www.st.com/en/evaluation-tools/32l496gdiscovery.html)
    8 is an evaluation board with the ARM Cortex-M4 based ultra-low power
    9 microcontroller STM32L496AG with 320KB of RAM and 1MB of ROM Flash.
    10 
    11 The main features of this board are:
    12 - 1.54 RGB 240 x 240 pixel TFT color LCD display with capacitive touch screen,
    13 - SAI audio codec, with stereo output, including analog microphone input
    14 - Stereo digital microphones
    15 - 8-Mbit external PSRAM
    16 - 64-Mbit external QSPI flash
    17 - USB OTG FS port
    18 - 8-bit camera interface
    19 - Stereo headset jack
    20 - micso-SD card connector
    21

    146 ## Flashing the device
    147 
    148 The STM32L496G-DISCO board includes an on-board ST-LINK programmer and can be
    149 flashed using OpenOCD. The board can be flashed with:
    150 
    151 ```
    152 make BOARD=stm32l496g-disco flash
    153 ```
    154 
    155 and debug via GDB with
    156 ```
    157 make BOARD=stm32l496g-disco debug
    158 ```
