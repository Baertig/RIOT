    Checks if path is a file or a directory.

    This function uses vfs_stat(), so if you need vfs_stat() anyway, you should not do double work and check it yourself.

    Parameters
    [in]pathPath to check
    Returns

    0 if path is a file
    > 0 if path is a directory
