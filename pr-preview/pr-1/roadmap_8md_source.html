    124 ## 802.15.4 link layer security
    125 (contact/steering: [chrysn](https://github.com/chrysn))
    126 
    127 Current status: RIOT supports application provided keys,
    128 with no guidance on how to (and no practical ways to) use that securely
    129 (see [CVE-2021-41061](https://nvd.nist.gov/vuln/detail/CVE-2021-41061)).
    130 
    131 Goal: Usably secure defaults.
    132 
    133 - Figure out applicability of [RFC9031](https://www.rfc-editor.org/rfc/rfc9031) ("CoJP") to non-6TiSCH scenarios.
    134 - Implement RFC9031 with any extensions needed for the MACs RIOT has.
    135 - Provide tools to set up a recommended JRC, and to provision keys between it and the device at flash time.
    136  This may entail extensions to the build process, as CoJP requires per-device secrets.
