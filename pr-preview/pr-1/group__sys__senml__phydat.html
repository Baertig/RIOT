    const phydat_t *  phydat,
    const uint8_t  dim 
    )

    Create a SenML decimal fraction value.

    Writes the value of the given dim of phydat as a decimal value. The unit of phydat is converted using the following rules:

    • UNIT_TIME is converted to SENML_UNIT_SECOND.
    • UNIT_BAR is converted to SENML_UNIT_PASCAL.
    • UNIT_GPM3 is converted to SENML_UNIT_KILOGRAM_PER_CUBIC_METER.
    • UNIT_GAUSS is converted to SENML_UNIT_TESLA.
    • Compatible units are set to their SenML equivalent.
    • Incompatible (or unknown) units are set to SENML_UNIT_NONE.
    Parameters
    [out]senmlSenML value to store value in.
    [in]phydatPhydat to convert.
    [in]dimDimension of phydat to convert.

    Create a SenML float value.

    Writes the value of the given dim of phydat as a float. The unit of phydat is converted using the following rules:

    • UNIT_TIME is converted to SENML_UNIT_SECOND.
    • UNIT_F is converted to SENML_UNIT_KELVIN.
    • UNIT_G_FORCE is converted to SENML_UNIT_METER_PER_SQUARE_SECOND.
    • UNIT_BAR is converted to SENML_UNIT_PASCAL.
    • UNIT_GPM3 is converted to SENML_UNIT_KILOGRAM_PER_CUBIC_METER.
    • UNIT_GAUSS is converted to SENML_UNIT_TESLA.
    • Compatible units are set to their SenML equivalent.
    • Incompatible (or unknown) units are set to SENML_UNIT_NONE.
    Parameters
    [out]senmlSenML value to store value in.
    [in]phydatPhydat to convert.
    [in]dimDimension of phydat to convert.
