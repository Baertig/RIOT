    Detailed Description

    Inverse Semaphore definitions.

    Inverse Semaphores can be used to synchronize on multiple threads / objects.

    The inverse semaphore can be used eiher in counter or in mask mode.

    In this mode the inverse Semaphore is initialized with a counter variable n. After n calls to sema_inv_post, the waiting thread is unblocked.

    In this mode the inverse Semaphore is initialized with a bit mask n. A call to sema_inv_post_mask clears one or multiple bits. Clearing the same bit multiple times has no effect. The thread is unblocked if all bits have been cleared.
